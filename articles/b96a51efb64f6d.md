# Upgrading Using AI

もし Laravel Boost でアップグレードを自動化したいときは、 `/upgrade-laravel-13` のコマンドを Agent になげればやってくれるみたいです。

## [High] Updating Dependency , [High] Updating Laravel Installer

composer.json でアップグレードが必要なパッケージは四つあります。

- laravel/framework to ^13.0
- laravel/tinker to ^3.0
- phpunit/phpunit to ^12.0
- pestphp/pest to ^4.0

もし Laravel installer を使っているときは、それも `composer global update laravel/installer` でアップグレードしてください。

## [High][New][Upgrade] Request Forgery Protection

Laravel CSRF Middleware （ **VerifyCsrfToken** ）の名前が変わり、Request Forgery Protection( **PreventRequestForgery** )になりました。

[**Sec-Fetch-Site** header -- developer.mozilla.org](https://developer.mozilla.org/ja/docs/Web/HTTP/Reference/Headers/Sec-Fetch-Site) を使うようになります。


```php
use Illuminate\Foundation\Http\Middleware\PreventRequestForgery;
use Illuminate\Foundation\Http\Middleware\VerifyCsrfToken;
 
// Laravel <= 12.x
->withoutMiddleware([VerifyCsrfToken::class]);
 
// Laravel >= 13.x
->withoutMiddleware([PreventRequestForgery::class]);
```
